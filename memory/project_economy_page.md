Built economy.html + economy.css as a standalone conversion-focused landing page for the Jetsetter Economy subscription product.

**Why:** User wants a modern, high-converting page to sell the $59.99/year Economy flight deal newsletter subscription.

**How to apply:** The Stripe checkout link for Economy is https://buy.stripe.com/aEUbKC9S22f2h1u7sC. Design uses dark/black background (#0a0a0a), deep red (#8B0000) as primary accent, white/light grey text. All assets live in "TCJ Website Assets/" folder. The skyline_headshot.jpg is used for the About Graeme section. The page has 12 sections: sticky nav, hero, why, how it works, past deals gallery (auto-scrolling CSS animation), pricing card, testimonials, about Graeme, press logos, social proof bar (red background), final CTA, footer.
